About PrivacyPal

Our Mission

PrivacyPal is your dedicated personal digital security companion. In an increasingly interconnected world, Our mission is to empower individuals, regardless of their technical background, with the knowledge and tools they need to navigate the digital landscape safely and confidently.

Through accessible educational resources and easy-to-use practical tools, we aim to demystify digital security and make it achievable for everyone. This project serves as a showase of web front-end design and development, emphasizing user experience and client-side data management.

Key Technologies

  • HTML5: For structuring all content, ensuring semantic and accessible markup.
  • CSS3: For modern presentation, responsive layouts, and a fluid user experience across all devices.
  • JavaSript (Vanilla JS & jQuery): For all interactive behaviours, dynamic content loading, and the core logic behind the security features (e.g., encryption, password generation). jQuery is used for efficient DOM manipulation.
  • CodeHS Firebase(Firestore Database): To dynamically store and retrieve educational articles, quiz questions, and to demonstrate basic user data interaction by storing quiz scores.
  • Git & Github: Used for robust version control, collaborative project management, and hosting the entier project repository.
